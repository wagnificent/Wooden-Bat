# Wooden Bat Classic Website

A modern, responsive website for the annual Wooden Bat Classic adult baseball tournament.

## Project Structure

```
wooden-bat-classic/
├── index.html                # Homepage
├── join-us.html              # Registration Hub
├── the-classic.html          # Tournament History & Legacy
├── tournament-hub.html       # Seasonal Updates & Standings
├── css/
│   └── styles.css            # Main CSS styles
├── js/
│   └── main.js               # JavaScript functionality
├── images/                   # All website images
└── README.md                 # Project documentation
```

## Features

### Design & Aesthetic
- **Theme**: Timeless Baseball Classic
- **Color Scheme**: Deep Navy Blue, Creamy Off-White/Ivory, Muted Red/Maroon
- **Typography**: Playfair Display (headings), Montserrat (body text)
- **Responsive Design**: Mobile-first approach with full responsiveness

### Pages
1. **Homepage** - High-impact landing page with hero section, key stats, and testimonials
2. **Join Us** - Registration hub with tabs for registration info, rules, and travel details
3. **The Classic** - Tournament history, legacy story, timeline, and photo archive
4. **Tournament Hub** - Latest updates, schedules, standings, and countdown timer

### Key Features
- **Interactive Tabs**: For organizing content on registration and tournament hub pages
- **Photo Archive Filtering**: Filter team photos by year and division
- **Countdown Timer**: Real-time countdown to tournament start
- **Mobile Navigation**: Responsive hamburger menu for mobile devices
- **Form Validation**: Newsletter subscription form with validation
- **SEO Optimized**: Proper meta tags, semantic HTML, and accessibility features

## Technical Stack
- **HTML5**: Semantic structure with proper heading hierarchy
- **CSS3**: Modern styling with CSS variables, flexbox, and grid
- **JavaScript**: Vanilla JS for interactivity (no framework dependencies)
- **Responsive Design**: Mobile-first approach with media queries
- **Accessibility**: ARIA labels, semantic HTML, and keyboard navigation

## Setup & Usage

1. **Local Development**:
   - Simply open any HTML file in a browser
   - No server or build process required

2. **Deployment**:
   - Upload all files to any web hosting service
   - No backend or database required

3. **Customization**:
   - Update content in HTML files
   - Modify styles in `css/styles.css`
   - Replace placeholder images in `images/` directory
   - Update tournament dates and details as needed
